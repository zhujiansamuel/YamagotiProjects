# GPU 加速相关依赖

# 注意：需要根据 CUDA 版本选择合适的 cupy 版本

# CUDA 11.x (例如 CUDA 11.8)
# cupy-cuda11x>=12.0.0

# CUDA 12.x (例如 CUDA 12.1)
# cupy-cuda12x>=12.0.0

# 如果不确定 CUDA 版本，可以使用：

# 1. cupy 包会自动检测系统中的 CUDA 版本

# 3. 生产环境建议使用具体的 CUDA 版本包（如 cupy-cuda12x）以获得更好的性能

# 对于 GPU 服务器：
#   pip install -r requirements-gpu.txt
